🧬 Compile Target Image → .mind

This helper converts a target image into the compact tracking file MindAR needs at runtime. Everything runs locally in your browser — no upload, no server.

How to use

  1. Click the zone below (or drag an image onto it).
  2. Click Compile and wait ~30 seconds.
  3. Click Download targets.mind.
  4. Move the downloaded file to assets/targets/targets.mind.

For best tracking, use a sharp, high-contrast image with lots of detail (text, patterns, corners). Plain colors or blurry photos will not track.

Tip: Want to try the sample chromosome image that ships with this project? Download assets/targets/chromosome-target.png and drop it here.
